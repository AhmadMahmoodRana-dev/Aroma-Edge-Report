# Aroma Edge Final Report — How to Export to PDF

## Step 1: Open the Report
Double-click `AromaEdge_Final_Report.html` to open it in your browser (Chrome recommended).

## Step 2: Print to PDF
1. Press **Ctrl + P** (or right-click → Print)
2. In the Print dialog:
   - **Destination:** Save as PDF
   - **Paper size:** A4
   - **Margins:** None (or Minimum)
   - **Scale:** 100%
   - ✅ Enable **Background graphics** (very important for colours and design!)
3. Click **Save** and name it: `AromaEdge_Final_Report.pdf`

## Step 3: Before Submission
Replace all [Placeholder] fields in the HTML file:
- [ ] Team member names
- [ ] Supervisor name
- [ ] LinkedIn profile links
- [ ] Social media links (Facebook, Instagram, YouTube, WhatsApp number)
- [ ] Website URL
- [ ] Campaign metrics (replace [X,XXX] placeholders with real numbers from your Meta Ads)
- [ ] Insert actual screenshots in the designated screenshot placeholders

## Tips
- Use **Google Chrome** for best PDF output
- Make sure background graphics is ON or the gold/dark design won't print
- Set margins to "None" for edge-to-edge design reproduction
